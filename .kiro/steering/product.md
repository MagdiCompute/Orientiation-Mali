# Product: Orientation Mali

A web-based school orientation application for Malian students taking the DEF (Diplôme d'Études Fondamentales) or BAC (Baccalauréat) exams.

## Purpose

Helps students make informed decisions about their academic future by:
1. Collecting responses to a 10-question profiling questionnaire
2. Analyzing responses with an AI Profile Agent to build a student profile
3. Generating personalized recommendations (majors, training programs, schools, career paths) via a Recommendation Agent

## Key Characteristics

- All user-facing content is in French
- Recommendations are specific to the Malian education system
- Two-agent architecture: Profile Agent → Recommendation Agent
- Linear synchronous flow: questionnaire → profile → recommendations → display
- Target users: Malian students at DEF or BAC level
